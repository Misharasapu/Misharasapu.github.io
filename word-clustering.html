Word Clustering and Nonlinear Classification

Two-part AI coursework. Task one builds unsupervised clusters of frequent words from Gothic fiction using hierarchical agglomerative clustering. Task two explores nonlinear decision boundaries on synthetic data, comparing logistic regression with a shallow neural network.

View repo View report (PDF)

Problem

Discover structure in text without labels and then compare linear versus nonlinear classifiers on a controlled dataset. The goal is to show when simple models are sufficient and when flexible function classes are needed.

Approach

  • Word clustering: built a word co-occurrence matrix from Gothic fiction chapters, then applied hierarchical agglomerative clustering with an appropriate distance linkage to form semantic groups. Visualised with dendrograms.
  • Nonlinear classification: generated a two-class synthetic dataset with curved class boundaries. Trained logistic regression and a small feed-forward neural network with regularisation and early stopping.
  • Evaluation: assessed cluster interpretability qualitatively and compared classifiers with standard accuracy metrics and decision boundary plots.

Results

  • HAC produced coherent word groups that reflected character names, settings, and thematic vocabulary from the corpus.
  • The neural network model recovered the curved boundary and achieved higher accuracy than logistic regression on the nonlinear dataset.
  • Logistic regression remained competitive on more linearly separable configurations and provided interpretable coefficients.

Tech stack

Python, NumPy, pandas, scikit-learn (HAC, logistic regression, MLPClassifier), matplotlib. Reproducible notebooks and fixed seeds.
